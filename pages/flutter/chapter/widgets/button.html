Button 사용하기

Flutter의 button은 크게 3가지로 나뉜다.ElevatedButton, TextButton, OutlinedButton이다.

ElevatedButton

Elevated - 높이가 있는 버튼 

                 
ElevatedButton(
    onPressed: () {},
    child: Text('ElevatedButton'),
    ),

Button 스타일 수정하기

ElevatedButton의 스타일을 수정하려면 style 속성을 사용한다.

                
style: ElevatedButton.styleFrom(),

padding 추가하기

                
padding: EdgeInsets.all(20.0),

Button 크기 수정하기

                
fixedSize: Size(300, 80),

Button 글자 크기 수정하기

                
textStyle: TextStyle(fontSize: 20, fontWeight: FontWeight.bold),

OutlinedButton

Outlined - 테두리가 있는 버튼

TextButton

Text - 텍스트만 있는 버튼
